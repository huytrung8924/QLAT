# ERO Riverside Website

Website giới thiệu dự án bất động sản ERO Riverside.

## Công nghệ sử dụng

- React
- Vite
- CSS

## Chức năng chính

- Trang giới thiệu dự án
- Danh mục sản phẩm (Liền kề, Shophouse, Biệt thự)
- Lọc và sắp xếp sản phẩm theo giá, diện tích
- Trang chi tiết sản phẩm với gallery ảnh
- Bản đồ Google Maps và sơ đồ dự án
- Thư viện hình ảnh và video
- Form đăng ký khách hàng
- Admin dashboard quản lý sản phẩm, bản đồ và gallery

## Dữ liệu hiện đang quản lý bằng LocalStorage chưa xây dựng Backend

## Tài khoản admin

- Username: admin
- Password: admin

## Chạy dự án

```bash
npm install
npm run dev

```
